# DesarrolloWeb
Código para el TFG- 2º DAW

Intercambio de datos con clientes mediante graficos.

Entorno de desarrollo : Laravel
Lenguajes de programacion: PHP, HTML, CSS

Visual:

Pagina Principal, Zona Admin, Zona Cliente.

El administrador crea a los usuarios y crea gráficos, por lo que los usuarios creados no se registran simplemente se loguean.
Los clientes tambien crean sus graficos.
